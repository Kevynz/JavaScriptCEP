Hoisting

Hoisting é o comportamento em que declarações de variáveis e funções são "elevadas" ao topo de seu escopo.

Exemplo de Código:


console.log(minhaVar); // undefined
var minhaVar = 5;

Arrow Functions

Arrow Functions são uma maneira concisa de declarar funções, apresentando diferenças importantes em relação às funções normais.

Exemplo de Código:


const soma = (a, b) => a + b;

Desestruturação (Destructuring)

A desestruturação permite extrair valores de arrays e objetos de maneira concisa.

Exemplo de Código:


const [a, b] = [10, 20];
const { nome, idade } = { nome: 'João', idade: 30 };

Clojure

Clojure é uma linguagem de programação funcional que enfatiza a imutabilidade e a simplicidade.

Exemplo de Código:


;; Exemplo de função simples em Clojure
(defn soma [a b]
  (+ a b))
